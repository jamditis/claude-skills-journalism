09 / Installation

Get Started

For Claude Code users

# Clone to your Claude skills directory
git clone https://github.com/jamditis/claude-skills-journalism.git ~/.claude/skills/journalism-skills
# Or install individual skills
cp -r source-verification ~/.claude/skills/

Skills load automatically when relevant to your work. For example, asking Claude to verify a source will activate the source-verification skill.
